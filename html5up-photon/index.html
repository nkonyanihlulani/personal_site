Movie Recommender Engine

For this project we used natural language processing techniques to prepare the data: we cleaned the data, and extracted features using both count vectorizer and tf-idf vectorizer to map our documents (tweets) into vectors. We fitted 9 different models and found that logistic regression (with count vectorizer) produced the best results. We then stacked the top 3 performing models and this improved our model.

Golden delicious apple price prediction using multiple regression techniques

This was a group project, we fitted 8 different regression models : linear, ridge, lasso, support vector, decision tree, random forest, xgboost, lightboost. Xgboot outperformed all the other models, we further tuned the hyperparameters until we found a root-mean-squared-error of 0.358 on the test set.
